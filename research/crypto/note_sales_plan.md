## 1. 全体戦略

### 1.1 ビジョン

AI を活用した予測・トレーディング技術を、**技術チュートリアル**と**リサーチ共有**という切り口で note.com 上にコンテンツ化し、以下の 3 段階で収益化する。

### 1.2 収益化の 3 ステップ

| フェーズ | 期間 | 目的 | 主要施策 |
|---------|------|------|---------|
| Phase 1: 集客期 | 月 1 - 3 | フォロワー獲得・信頼構築 | 無料記事 10 本 + Twitter/X 連携 |
| Phase 2: 単品販売期 | 月 4 - 6 | 初収益・価格検証 | 有料記事 5-7 本 + 有料マガジン 2 種 |
| Phase 3: サブスク期 | 月 7 以降 | 安定月額収益 | 定期購読マガジン + コミュニティ |

### 1.3 KPI マイルストーン

| 指標 | 3 ヶ月目 | 6 ヶ月目 | 12 ヶ月目 |
|------|---------|---------|----------|
| フォロワー数 | 300 | 1,000 | 3,000 |
| 月間 PV | 5,000 | 20,000 | 60,000 |
| 月間収益 | 0 円 | 30,000 円 | 150,000 円 |
| 有料記事購入者累計 | 0 | 50 | 300 |
| 定期購読者数 | 0 | 0 | 100 |

### 1.4 note.com を選ぶ理由

- 技術系・投資系の読者層が厚い
- 有料記事/マガジン/定期購読の仕組みが標準装備
- SEO に強く Google 検索からの流入が期待できる
- 手数料体系が明確（記事販売 15%、定期購読 ~25%）

## 4. マーケティング戦略

### 4.1 Twitter/X アカウント戦略

**アカウント設計:**
- ハンドル名: AI 予測・Python 関連のわかりやすい名前
- プロフィール: 「AI × 投資予測の研究記録。Python / 機械学習 / 公営競技 / 暗号通貨」
- 固定ツイート: 最も PV が高い無料記事へのリンク

**投稿戦略:**

| 投稿タイプ | 頻度 | 内容 |
|-----------|------|------|
| 記事宣伝 | 記事公開時 | タイトル + 要約 + リンク + アイキャッチ画像 |
| 予測結果共有 | 週 2-3 回 | 「今週のAI予測結果」グラフ画像付き |
| 技術 Tips | 毎日 | Python / ML の小ネタ（コードスニペット画像） |
| 実況系 | 不定期 | 「今日のボートレース、AIの予測は...」リアルタイム感を演出 |
| 数字報告 | 月 1 回 | 「月間予測精度 XX%」等のパフォーマンス報告 |

**フォロワー獲得施策:**
- AI / Python / 投資系アカウントと積極的に交流
- 技術系ハッシュタグ（#Python, #機械学習, #AI予測）を活用
- 予測結果の画像は「いいね」「RT」されやすいフォーマットで統一
- Qiita / Zenn にも同テーマの記事を投稿し、note.com へ誘導

## 5. 法的注意事項

### 5.1 必須免責事項（全記事共通）

以下の免責事項を**全ての記事**（無料・有料問わず）の冒頭と末尾に掲載する。

```
【免責事項】
本記事は技術研究・教育目的で作成されたものであり、
投資助言、投資勧誘、または特定の金融商品の推奨を行うものではありません。

・投資に関する最終的な判断は、ご自身の責任において行ってください。
・本記事に掲載された予測結果やバックテスト結果は過去のデータに基づくものであり、
  将来の運用成果を保証するものではありません。
・本記事の内容に基づいて生じたいかなる損失についても、筆者は一切の責任を負いません。

※ 公営競技（ボートレース・競輪・競馬）の予測は娯楽・研究目的です。
※ 暗号資産・FX・株式の予測は技術的な研究結果の共有であり、売買の推奨ではありません。
```

### 5.2 記事の表現ルール

**絶対に避けるべき表現:**
- 「必ず儲かる」「確実に利益が出る」「損しない」
- 「月収 XX 万円達成」（実際の取引利益として）
- 「この手法で勝率 100%」
- 「元本保証」「リスクゼロ」

**推奨する表現:**
- 「研究の結果、バックテストでは XX% の精度でした」
- 「あくまで技術的な検証結果です」
- 「実運用には別途リスク管理が必要です」
- 「教育・学習目的のコンテンツです」

### 5.3 ソースコード販売に関する注意

- 「このコードを使えば儲かる」という表現は厳禁
- 「プログラミング学習教材」「技術チュートリアル」として位置づける
- コードの動作保証はしない旨を明記
- 外部 API（取引所 API 等）の利用規約を遵守する旨を注記

### 5.4 公営競技に関する追加注意

- 予想販売業として営業する場合、地方競馬は登録が必要な場合がある
- 「予想」ではなく「AI予測技術の研究結果共有」として位置づける
- 舟券・車券・馬券の購入を直接促す表現は避ける

### 5.5 金融商品取引法への配慮

- 投資助言業の登録なしに具体的な売買推奨は行わない
- 「この通貨ペアを買え」「この銘柄を売れ」という表現は厳禁
- あくまで「AIモデルの出力結果」「技術的な研究」として共有
- 有価証券報告書の記載内容等を引用する場合は出典を明記
