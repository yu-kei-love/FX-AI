## 第1章: 基本戦略 — 売れるまでのロードマップ

### Rule 1: いきなり有料記事を出すな
- 収益化までの目安は **3〜6ヶ月**
- 最初の1〜2ヶ月は無料記事のみで信頼構築
- 無名クリエイターの有料記事が偶然売れることはない

### Rule 2: 無料記事→有料記事の導線を設計せよ
- 無料記事の目的: 認知拡大・フォロワー獲得・ファン化
- 有料記事の目的: 収益化・熱心な読者への深い価値提供
- 無料記事の末尾で関連する有料記事を自然に紹介する

### Rule 3: プロフィール1ページ目に有料記事を常時配置せよ
- プロフィール画面と同時に表示される1ページ目に有料記事を複数配置
- 新規訪問者が即座に商品を見つけられるようにする

### Rule 4: 月3〜5本を目安に投稿を継続せよ
- 更新頻度より質の安定が重要
- 「この人の発信なら対価を払ってでも読みたい」という関係性を構築

## 第2章: 有料記事の構成 — 無料部分で9割決まる

### Rule 5: 無料部分は「セールスレター」と心得よ
note編集部が有料記事100件を分析した結果、購入される記事の無料エリアには以下の3要素が含まれる:
- **(A) 共感・問いかけ**: 読者の悩みを言語化して共有
- **(B) 変化の物語**: Before/Afterの経験を示す
- **(C) ベネフィット提示**: 読むことで得られる価値を明示

### Rule 6: PASONAの法則を使え
1. **Problem（問題提起）**: 「FXで勝てなくて悩んでいませんか？」
2. **Agitation（煽り）**: 「裁量トレードでは感情に負けて損切りできない…」
3. **Solution（解決策）**: 「AIが感情を排除して判断します」
4. **Narrow down（絞込）**: 「このモデルはWalk-Forward検証済み」
5. **Action（行動喚起）**: 「この先で具体的な方法をすべてお伝えします」

### Rule 7: 無料部分の目次は「学べること」まで書け
- 300〜500文字で目次を掲載
- 各章で何が学べるか、どんなテンプレートが付いているかまで明記
- ただの目次ではなく、ベネフィットリストにする

### Rule 8: 機能ではなくベネフィットを書け
- NG: 「CatBoostで特徴量200個を学習したモデルです」
- OK: 「バックテストで年間リターン+XX%を達成した手法の全てをお見せします」
- 読者が欲しいのは技術の詳細ではなく「その先にある理想の未来」

## 第3章: タイトル戦略

### Rule 9: タイトルは25文字以内を目指せ
- X（Twitter）上で長いタイトルは「...」と省略される
- 大事なキーワードを文頭に置く
- SNS表示時に切れない長さに収める

### Rule 10: クリックされるタイトルの7要素
1. **数字を入れる**: 「月利+12%を実現した」「3つの法則」
2. **固有名詞を入れる**: 「CatBoost」「Walk-Forward」
3. **ターゲットを明示**: 「FX初心者が」「副業トレーダー向け」
4. **限定感**: 「2026年最新版」「今だけ公開」
5. **カッコ書き**: 「【検証済】」「【実績公開】」
6. **結論を示す**: 内容が分かる方がクリックされる
7. **損失回避ワード**: 「知らないと損する」「やってはいけない」

### Rule 11: 期待させすぎるな
- タイトルで過剰な期待を煽ると、読後の満足度が下がる
- リピーターが生まれず、長期的な売上に悪影響
- 「必ず稼げる」は景品表示法にも抵触しうる

## 第4章: 価格設定

### Rule 12: 段階的に値上げせよ
- 初期は低価格（500〜1,000円）でスタート
- 読者の反応・評価を見ながら段階的に上げる
- noteの実用ノウハウ系の平均価格は **1,842円**

### Rule 13: 手数料を理解せよ
- プラットフォーム利用料: **売上の10%**
- 決済手数料: クレジットカード **5%**、携帯キャリア決済 **15%**
- 振込手数料: 1回あたり **270円**
- 500円の記事 → 手取りは約 **425円**（クレカ決済の場合）

### Rule 14: 高額記事には実践ツールを付けよ
- 10,000円以上の高額記事には以下を付加:
  - Excelの計算シート / Pythonスクリプト
  - チェックリスト / テンプレート
  - プロンプト集 / 設定ファイル
- 「読んで終わり」ではなく「すぐに使える」を提供

## 第5章: 収益モデルの選択

### Rule 15: 7つの収益モデルを使い分けよ
1. **有料記事（単発）**: AI予測の手法解説、バックテスト結果
2. **有料マガジン**: 複数記事のセット販売（単品合計の50〜80%に設定）
3. **定期購読マガジン（月額）**: 毎日/毎週のAI予測配信 → 安定収益の柱
4. **メンバーシップ**: コミュニティ型（質問対応・限定情報）
5. **サポート（投げ銭）**: 無料記事への感謝課金
6. **ストア販売**: Pythonスクリプト、EAファイルなど
7. **Amazonアフィリエイト**: 関連書籍の紹介（noteで使える唯一のASP）

### Rule 16: 定期購読で安定収益を作れ
- 月額課金 = 毎月の固定収入
- AI予測は「継続的な情報更新」と相性が極めて良い
- 競馬AI予想家はnoteの定期購読モデルで成功している例が多い

## 第6章: SNS集客（X/Twitter特化）

### Rule 17: X（Twitter）が最重要集客チャネル
- noteへの外部流入で最も多いのはX経由
- Xのリツイート機能は拡散性で他SNSを圧倒
- Instagram/TikTokからWebページへの誘導は難しい

### Rule 18: X投稿の基本ルール
- **1日1回以上**の投稿を目標
- **朝の通勤時間・昼休み・夜のリラックスタイム**に投稿
- 画像付きツイートはCTRが **18%向上**（HubSpot調査）
- 適切なハッシュタグを使用: #FX #AI予測 #自動売買 #ボートレース

### Rule 19: SNSでは記事の「結果」を見せよ
- 「今日のAI予測が的中しました！詳細はnoteで」のパターン
- 的中実績・リターン実績をグラフや画像で投稿
- 無料で実績を見せ続けることが信頼の源泉

### Rule 20: 口コミ（UGC）を記事に埋め込め
- 読者のポジティブなツイートをnote記事内に貼り付け
- 第三者の評価が記事の信頼性を大幅に高める

## 第7章: 法的注意事項 — 絶対に守れ

### Rule 21: 禁止表現を理解せよ
以下の表現は **景品表示法・消費者契約法違反** のリスクあり:
- 「必ず稼げる」「確実に利益が出る」 → **優良誤認表示**
- 「今だけ特別価格」（実際は常時その価格） → **有利誤認表示**
- 「簡単に月収100万円」 → **不実告知**
- 利益を保証する表現は全てNG

### Rule 22: 適切な表現例
- OK: 「バックテストでは年間リターン+XX%でした（将来の利益を保証するものではありません）」
- OK: 「過去X年間のWalk-Forward検証結果をお見せします」
- OK: 「あくまで参考情報であり、投資判断はご自身でお願いします」
- **必ず免責事項（ディスクレーマー）を記載する**

### Rule 23: 投資助言業に該当しないよう注意せよ
- 「この通貨を買え/売れ」という具体的な売買指示 → 投資助言業に該当しうる
- 「AIモデルの構築方法」「分析手法の解説」→ 情報提供であり問題なし
- 「予測結果の参考情報」として提供し、投資判断は読者に委ねる
- 不安な場合は弁護士に相談すること

### Rule 24: 不実証広告規制（15日ルール）
- 商品の効果・性能について、消費者庁から根拠資料を求められた場合
- **15日以内** に合理的な根拠を提出できなければ違法扱い
- バックテスト結果、Walk-Forward検証結果などの根拠を常に保持せよ

### Rule 25: ステルスマーケティング禁止
- 2023年10月からステマは景品表示法違反
- 他者に宣伝を依頼する場合は「PR」「広告」と明記必須
- 自作自演のレビューもNG

## 第8章: AI予測ニッチの競合分析と差別化

### Rule 26: 競馬AI市場の現状を知れ
- noteで「競馬予想」で検索すると **約4,000の販売者** が存在
- 価格帯: 100〜10,000円
- 成功者の共通点: SNSフォロワーが多い、無料予想で実績を示している
- AIウマスギなどの人気アカウントは、開発費500万円を半年で回収と主張
- ただし品質のばらつきが大きく「お小遣い欲しさに売っている」レベルの販売者も多い

### Rule 27: FX AI市場の現状を知れ
- ChatGPT連携のFX自動売買EA開発記事がnoteで注目されている
- GPT-EA（生成AI搭載型FX自動売買ツール）の無料テンプレート提供が人気
- 「AIで簡単に稼げる」系の詐欺商品も多く、差別化には「検証の透明性」が鍵

### Rule 28: ボートレースAI市場はブルーオーシャン
- note上でのボートレースAI予想販売者は競馬に比べて少ない
- 外部にはBOATERS、梅吉AI、ポセイドンなどのサービスが存在
- 「ボートレース × AI × note」は参入余地あり

### Rule 29: 差別化の3要素
1. **検証の透明性**: Walk-Forward検証、PBO（Probability of Backtest Overfitting）の結果を公開
2. **実体験の深さ**: 2026年はAI生成の無料情報が溢れているため「実体験に基づくリアルな情報」の価値が上昇中
3. **再現性の提供**: 読者が自分で再現できるコード・手順・データを提供

## 第9章: 具体的なコンテンツ企画案

### 企画A: 単発有料記事シリーズ
1. 「FX AI予測モデルの作り方 — CatBoost + Walk-Forward完全ガイド」（3,000〜5,000円）
2. 「ボートレースAI予測モデル構築記 — 回収率XX%の裏側」（2,000〜3,000円）
3. 「バックテストの罠 — 補間データが生む幻の好成績」（1,000〜2,000円）
4. 「Sharpe比の正しい計算方法 — 全バーで計算しないと意味がない」（500〜1,000円）

### 企画B: 定期購読マガジン
- 「AI予測ウィークリーレポート」（月額980〜1,980円）
  - 毎週のモデル予測結果と振り返り
  - ペーパートレード実績の公開
  - モデル更新ログ

### 企画C: 高額パッケージ
- 「AI予測システム構築キット」（10,000〜30,000円）
  - Pythonスクリプト一式
  - 特徴量設計ドキュメント
  - セットアップガイド
  - 1ヶ月間のメール質問対応

### 企画D: 無料記事（集客用）
- モデル開発の失敗談（補間データの教訓など）
- 週次の予測的中/外れレビュー
- AI/ML初心者向けの基礎解説
