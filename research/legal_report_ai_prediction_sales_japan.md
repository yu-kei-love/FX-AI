## 結論サマリー

**AI予測コード（プログラムのソースコード）の販売は基本的に合法である。** ただし、以下の条件を満たす必要がある：

| 項目 | 判定 |
|------|------|
| 競馬・競艇等の予想を有料で提供すること | **合法**（予想屋は法的に認められた職業） |
| AI予測のソースコードを販売すること | **合法**（ソフトウェア販売として扱われる） |
| 「必ず儲かる」「回収率100%保証」等の表現 | **違法**（誇大広告・消費者契約法違反） |
| 特定の株式銘柄の売買推奨 | **違法の可能性大**（投資助言業登録が必要） |
| 競馬・公営競技の予想販売 | **合法**（金融商品取引法の対象外） |
| noteでの投資・ギャンブル系有料記事 | **要注意**（規約上のグレーゾーン、BAN事例あり） |

## 法的フレームワーク

### 1. 予想販売の合法性（公営競技）

**公営競技（競馬・競艇・競輪・オートレース）の予想販売は、日本の法律上明確に合法である。**

根拠：
- 競輪・競艇・オートレース・地方競馬には「公認予想屋」制度が存在し、競技場から認可を受けて有料で予想を提供している
- 予想の販売は「情報の提供に対する対価」であり、一般的な情報販売ビジネスと同じ扱い
- 中央競馬（JRA）では場内の公認予想屋は認めていないが、インターネット上での予想販売自体は禁止されていない
- 弁護士ドットコムの複数の弁護士回答でも、予想販売自体の合法性が確認されている

**重要な区別：** 公営競技の予想は「金融商品」に該当しないため、金融商品取引法上の投資助言業登録は不要である。競馬の馬券は有価証券ではなく、デリバティブ取引でもない。

### 2. 金融商品取引法と投資助言業

**AI予測が「投資助言業」に該当するかどうかは、対象によって大きく異なる。**

#### 投資助言業に該当しないケース（登録不要）：
1. **不特定多数向けの情報提供** -- 金融庁の監督指針では「不特定多数の者を対象として、不特定多数の者が随時に購入可能な方法により投資情報等を提供する行為」は投資助言業に該当しないとされている
2. **ソフトウェアの販売** -- 「誰でも、いつでも自由にソフトウェアを購入できる状態にある場合」は登録不要。ただし、購入後に継続的な投資データの提供やサポートが必要な場合は該当する可能性あり
3. **無償での情報提供** -- 投資顧問契約は有償性が要件のため、無料ブログ等は対象外
4. **基礎資料レベルの情報提供** -- 景気動向や企業業績等の基礎情報の提供のみでは該当しない
5. **公営競技の予想** -- 有価証券でもデリバティブ取引でもないため対象外
6. **暗号資産（現物）の助言** -- 有価証券でもデリバティブでもないため対象外

#### 投資助言業に該当するケース（登録必要）：
1. 特定の株式銘柄の売買ポイントを具体的にアドバイスし、対価を得る場合
2. 会員登録しないと購入できないクローズドな形式での投資情報提供
3. 個別・相対性の高い投資情報の提供
4. ソフトウェア販売後に継続的な投資データ・サポートの提供がある場合

**罰則：** 無登録で金融商品取引業を営業した場合、5年以下の懲役若しくは500万円以下の罰金（併科あり）。近年、金融庁は取締りを強化中。

### 3. 特定商取引法（特商法）

noteで有料記事を継続的に販売する場合、特定商取引法上の「販売業者」に該当する可能性がある。

- **「業として営む」の定義：** 営利の意思をもって、反復継続して取引を行うこと
- **表記義務：** 氏名、住所、電話番号の掲載が原則（省略規定あり）
- **個人でも対象：** 客観的に「業として」判断されれば、個人でも特商法の事業者となる
- **noteの立場：** プラットフォームとしてクリエイター自身が法令に基づいた表示を行う必要がある旨を示している
- **省略規定：** 消費者から請求があった場合に「遅滞なく」開示できる体制があれば、掲載を省略可能

### 4. 消費者契約法・景品表示法

以下の表現は明確に違法：
- 「絶対に当たる」「100%的中」（断定的判断の提供）
- 「必ず儲かる」「損失ゼロ」（不実の告知）
- 実在しない実績データの捏造（虚偽表示）
- 「八百長情報がある」（詐欺罪に該当）

これらに基づく契約は「錯誤無効」となり、返金請求が可能。

## note.comの規約と実態

### noteの利用規約における禁止事項

noteの利用規約（第22版・2026年1月15日制定）で明確に禁止されているもの：

1. **「株式の銘柄推奨、その他金融商品取引法に抵触するもの」** -- 例：「2025年絶対買うべきは○○」のような表現
2. **「必ずもうかる」等、ユーザーに著しい誤解を招く表現を用いたもの**
3. **確定・未確定を問わず売上、利益その他これに類するものを公開して、コンテンツの購入を煽る行為**
4. **マルチ商法等、noteがユーザーに不利益をもたらすと判断する情報商材**

### noteの実際の運用・監視体制

- **Trust & Safetyチーム** が「機械的な検知」と「目視チェック」で不適切コンテンツを監視
- **要注意テーマとして「投資系・ギャンブル系」** が明確に挙げられている
- 「断定系表現」が確認された場合、事前警告なしで公開停止措置が行われる
- 違反時のペナルティ：記事の公開停止、アカウント停止、売上金没収、違約金請求

### noteで実際にBANされた事例

- 投資・相場系、お金儲け系の有料記事が「狙い撃ち」されている報告が複数あり
- テクニック的な有料記事を販売していたトレーダーが公開停止された事例
- 違法行為をしていないにもかかわらず、「規約違反」として売上金没収されたケース
- 処分に関する個別の質問にはnoteは応じておらず、どの部分が違反か自分で判断する必要がある
- 事前警告なしでいきなり公開停止になるため、対策が難しい

### noteの「情報商材」に関する公式見解

noteに直接問い合わせた事例によると、noteは「情報商材の販売」を禁止しているが、「情報商材」の定義が曖昧であり、個別のケースごとに判断している模様。ギャンブル系・投資系の有料記事は情報商材と判断されるリスクが通常より高い。

## 実際の販売事例

### note上で現在販売されている競馬AI関連コンテンツ

2026年3月時点で、note.comには多数の競馬AI関連の有料記事が存在する：

1. **「コピペで競馬AI開発」シリーズ** -- コードを全公開し、「回収率120%」を謳う有料記事。2026年1月にVer.2.0に更新
2. **「データサイエンス研究所」のマガジン** -- 競馬AIの全工程（データ収集、前処理、特徴量エンジニアリング、機械学習、自動投票）のコードを有料マガジンとして販売。単品合計71,980円程度
3. **「非エンジニアのための競馬AIの作り方」** -- 回収率150%を謳い、ChatGPTモデルやExcelファイル等の特典付き
4. **「馬券自動購入できる競馬AIアプリの作り方」** -- Pythonベースの馬券自動購入システム
5. **「競馬全自動購入ツールプログラム」** -- 地方競馬（SPAT4）専用の自動購入ツール。高額設定

### noteでの競馬予想販売の市場規模

- 2023年5月時点で「競馬予想」で検索すると約4,000の販売ユーザー
- 2026年2月時点で19名以上の競馬予想家がランキング化されている
- 有名予想家は数百人～数千人規模のフォロワーを持つ

## トラブル・摘発事例

### 法的に逮捕・摘発された事例

**注意：以下はいずれも「予想の販売」自体ではなく、「詐欺行為」で摘発されたケース。**

1. **千葉県・競馬情報会社事件** -- 「高額配当が得られる情報がある」と偽り、延べ約4万人から約29億円を集めた経営者ら13人が逮捕
2. **兵庫県・八百長情報詐欺** -- 「競馬の八百長レース情報を教える」とDMを送り、7つの口座に総額13億円超の入金。5人送検
3. **怒羅権メンバー詐欺** -- 「確定レースで必ず当たる」と偽り200万円詐取
4. **出資法違反事件** -- 競馬で高額配当するとうたい不正に資金を集めた投資顧問会社幹部を逮捕

### noteでの予想パクリ・盗用事例

- 弁護士ドットコムが2020年に報道：YouTuberが他の予想家の予想をnoteで盗用して有料販売
- 2025年1月：X（Twitter）の予想家「うまうま君」がnetkeiba公認予想家の予想を丸パクリした疑惑でアカウント削除

### note上での規約違反による公開停止

- 投資系の有料記事が「確実に効果があるといった趣旨の表現」を理由に公開停止された事例多数
- 売上金が没収され、異議申し立ての手段が限られるケース
- アカウント削除により利益を受け取れなくなった事例

## リスク評価

### 「AIの予測コード」を販売する場合のリスク

| リスク | 深刻度 | 説明 |
|--------|--------|------|
| 法的リスク（刑事） | **低** | 公営競技の予想は金商法の対象外。詐欺的表現をしなければ刑事リスクは極めて低い |
| 法的リスク（民事） | **中** | 「回収率○○%保証」等の表現で購入者が損失を被った場合、返金訴訟の可能性 |
| noteのBAN | **中～高** | ギャンブル系は要注意テーマ。断定的表現があると公開停止のリスクあり |
| 売上金没収 | **中** | アカウント停止時に売上金が没収される可能性 |
| 評判リスク | **低～中** | 予測が外れた場合のクレーム、悪評の拡散 |
| 特商法違反 | **低** | 継続販売の場合は表記義務あり。省略規定を活用可能 |
| 著作権リスク | **低** | 自分で書いたコードであれば問題なし。他者のコード流用は違法 |
| データ利用規約違反 | **中** | JRA-VAN等のデータ提供元の利用規約に抵触する可能性 |

### 「AI予測の結果（買い目）」を販売する場合のリスク

| リスク | 深刻度 | 説明 |
|--------|--------|------|
| 法的リスク | **低** | 予想販売自体は合法。ただし表現に注意 |
| noteのBAN | **高** | ギャンブル予想の直接販売はnoteで最も警戒されるカテゴリ |
| 返金トラブル | **高** | 予想が外れた場合の返金要求リスク |

## 安全に販売するためのベストプラクティス

### 1. コンテンツの位置づけ

**「予想の販売」ではなく「教育コンテンツ・技術コンテンツの販売」として明確に位置づける。**

- 「AI予測の買い目」を売るのではなく、「AI予測モデルの作り方」「プログラミングの学習教材」として販売
- 実際にnoteで成功している競馬AIコンテンツの多くは、この「教育コンテンツ」形式を採用
- コードの解説、技術的なノウハウ、データサイエンスの学習教材という形が最も安全

### 2. 絶対に避けるべき表現

以下の表現は法律違反・規約違反の両方に該当するため、絶対に使用しない：

- 「必ず儲かる」「絶対当たる」「100%的中」
- 「損失ゼロ」「リスクなし」
- 「○○万円稼いだ」（売上公開による購入煽り）
- 「回収率○○%保証」
- 「八百長情報」「インサイダー情報」
- 特定の金融商品（株式・FX等）の具体的な売買指示

### 3. 推奨する記載事項

記事内に必ず含めるべき免責事項：

- 「本コンテンツは教育・学習目的の技術情報です」
- 「馬券の購入は自己責任でお願いします」
- 「過去の回収率は将来の結果を保証するものではありません」
- 「本コンテンツは投資助言を行うものではありません」
- 「ギャンブルには依存性があります。適度に楽しみましょう」

### 4. noteでの安全な販売方法

1. **断定的表現を徹底排除** -- 「～の可能性がある」「～を検証してみた」等の表現を使用
2. **売上・利益の公開禁止** -- noteの規約で明確に禁止されている
3. **「教育・技術コンテンツ」として販売** -- 予想そのものではなく、予想手法・コード・技術を販売
4. **無料部分で十分な説明** -- 購入前に内容が理解できる構成にし、「説明と実態の乖離」を防ぐ
5. **特商法の表記** -- 継続販売する場合は、省略規定を活用しつつも表記を行う
6. **バックアップ先の確保** -- noteがBANされた場合に備え、Tips・Brain・Zenn等の代替プラットフォームを用意

### 5. プラットフォーム選択

| プラットフォーム | ギャンブル系の寛容度 | 特徴 |
|-----------------|---------------------|------|
| note.com | **低～中** | 上場企業のためクリーンさ重視。ギャンブル系は要注意テーマ |
| Tips | **中～高** | note移行組が多い。情報商材に比較的寛容 |
| Brain | **中～高** | 紹介報酬機能あり。情報商材向け |
| Zenn | **中** | 技術記事プラットフォーム。コード販売に適している |
| Gumroad | **高** | 海外サービスで規制が緩い。決済が容易 |
| BOOTH | **中** | デジタルコンテンツ販売に特化 |
| 自サイト | **最高** | 自分の裁量で運営可能。ただし集客は自力 |

### 6. 法的に最も安全な販売形態

**最も安全：** 「AI予測モデル開発のチュートリアル」「機械学習を使った公営競技データ分析入門」のような教育コンテンツとして、ソースコードを提供する形

**理由：**
- ソフトウェア（コード）の販売は、金融庁の監督指針で「不特定多数向けのソフトウェア販売」として投資助言業の適用除外に該当
- 公営競技の予想は金融商品取引法の対象外
- 教育コンテンツはnoteの規約上も問題が少ない
- 「予想を売る」のではなく「予想する方法を教える」形式なので、結果に対する責任が限定的
