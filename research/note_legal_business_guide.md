## 1. 金融商品取引法と投資助言業の基本

### 1.1 投資助言・代理業とは

金融商品取引法第28条3項により、以下の3要素がすべて揃った場合に「投資助言・代理業」に該当する:

1. **有価証券の価値等**または**金融商品の投資判断**に関する助言であること
2. **投資顧問契約**（助言を行い、相手方が報酬を支払う契約）を締結していること
3. これを**業として**行っていること

### 1.2 無登録営業の罰則

- 金融商品取引法第29条: 登録なしに金融商品取引業を営むことは違法
- **罰則**: 5年以下の懲役もしくは500万円以下の罰金、またはこれを併科
- 方法は問われない（有料メルマガ、オンラインサロン、note有料記事いずれも対象）

### 1.3 投資助言業の登録要件（参考）

登録する場合のコスト:
- 登録免許税: **15万円**
- 営業保証金: **500万円**（供託）
- その他: 十分な知識・経験を有する役員または使用人の確保

> 出典: [財務省関東財務局 - 投資助言・代理業関係](https://lfb.mof.go.jp/kantou/kinyuu/toushijogen/annai.htm)

## 2. 合法 vs 違法の境界線

### 2.1 登録不要（合法）となる条件

金融庁の監督指針に基づき、以下の場合は投資助言業の登録が**不要**:

#### (A) 不特定多数向けの情報提供（最重要）

> 「不特定多数の者を対象として、不特定多数の者が随時に購入可能な方法により、投資情報等を提供する行為」

**具体的な条件:**
- 誰でも、いつでも購入できる状態であること
- 会員限定・購入者限定にしないこと
- **単発購入を受け付けること**（月額課金のみは危険）

#### (B) 投資分析ツール・ソフトウェアの販売

- 誰でも自由に購入できる状態にあること
- ただし、購入後に継続的なサポートやデータ提供が必要な場合は登録が必要になる可能性あり

#### (C) 基礎資料的な情報提供にとどまる場合

- 景気動向、企業業績、市場統計データなどの提供
- 単なる円安/円高の値動き予想の配信
- ただし、特定銘柄の売買ポイント等の具体的助言は規制対象

#### (D) 報酬を受けない場合

- 無料での情報提供は投資助言業に該当しない

#### (E) 暗号資産の現物（2027年頃まで）

- ビットコイン等のユーティリティトークンの「現物」に関する助言は現時点では登録不要
- **注意**: 2027年以降は規制対象になる見込み

### 2.2 登録が必要（違法リスクあり）なケース

| ケース | リスク度 |
|--------|---------|
| 有料noteで「この銘柄は買い」と助言 | **高** |
| 読者からの相談に対して投げ銭を受けて投資判断を提示 | **高** |
| 有料マガジンで特定銘柄を継続的に推奨 | **高** |
| 月額課金制の投資情報メルマガ（単発購入不可） | **高** |
| 購入者限定Discord等での個別助言 | **高** |
| 「会員登録必須」で投資情報を購入・利用する形態 | **中〜高** |

### 2.3 グレーゾーンの判断基準

判断の3つの軸:
1. **内容**: 具体的な投資判断の助言に該当するか？
2. **対価**: 報酬を受け取っているか？
3. **アクセス**: 不特定多数が随時購入可能な状態か？

> 出典: [投資助言業に該当しないケース（サポート行政書士法人）](https://www.shigyo.co.jp/search_post/kinyu/toushijyogen/jyogen_kaitousinai/), [牛島総合法律事務所](https://www.ushijima-law.gr.jp/topics/20250421investment_advisory/), [金融庁監督指針](https://www.fsa.go.jp/common/law/guide/kinyushohin/07.html)

## 3. ジャンル別の法的整理

### 3.1 FX（外国為替）予測の販売

**結論: 構成次第で合法だが、最もリスクが高いジャンル**

- FXは金融商品取引法の規制対象
- 「USD/JPYを○○円で買い」は投資助言に該当する可能性が高い
- AIモデルの構築過程・教育的コンテンツとして提供すれば規制を回避しやすい
- 自動売買ツール（EA）の販売は、継続的サポートが不要なら登録不要の可能性

> 出典: [弁護士によるFXコンサル/サロンの適法性検証](https://fxatty.com/law4/)

### 3.2 競馬予測の販売

**結論: 基本的に合法**

- 競馬予想を有料で販売すること自体は違法ではない
- 情報の売買として一般的なビジネスに該当
- **違法になるケース:**
  - 「100%的中」「絶対に当たる」等の誇大広告（特定商取引法違反）
  - インサイダー情報の詐称（詐欺罪）
  - ノミ行為
  - 強引な金銭要求（ご祝儀の強要等）

> 出典: [競馬予想サイトの違法性（悪質競馬予想サイト調査局）](https://keiba89.com/keiba-report/illegal/)

### 3.3 競艇予測の販売

**結論: 基本的に合法（競馬と同様）**

- 競艇（ボートレース）の買い目予想の販売自体は情報販売の範囲内
- noteでは数百円〜数千円で多数の予想が販売されている
- 虚偽広告や的中保証表現を避ければ合法

> 出典: [弁護士ドットコム](https://www.bengo4.com/c_1009/n_11682/)

### 3.4 暗号資産（仮想通貨）予測の販売

**結論: 現時点では比較的自由だが、2027年以降は規制強化の見込み**

- 暗号資産現物に関する投資助言は現行法では登録不要
- ただし、暗号資産デリバティブ取引は既に規制対象
- 2027年以降は現物・デリバティブともに金融商品取引業として規制される可能性

### 3.5 株式予測の販売

**結論: 最もリスクが高い。noteでは銘柄推奨が明確に禁止**

- noteの利用規約で「株式の銘柄推奨」が明確に禁止されている
- 金融商品取引法の規制が直接適用される
- 教育コンテンツ（分析手法、ツール作成方法）に徹する必要がある

## 4. noteの利用規約と禁止事項

### 4.1 投資関連の明確な禁止事項

noteの利用規約で以下が明確に禁止されている:

1. **株式の銘柄推奨、その他金融商品取引法に抵触するもの**
2. **マルチ商法等、ユーザーに不利益をもたらすと判断する情報商材**
3. **売上・利益の公開による購入の煽り**
   - 確定・未確定を問わず売上・利益を公開してコンテンツの購入を煽る行為
4. **ユーザーに誤解を与える表現**
5. **センシティブな内容**

### 4.2 投資系で特に注意すべきポイント

- **断言系表現の禁止**: 「絶対に儲かる」「必ず利益が出る」等は即公開停止の対象
- **売上スクリーンショット**: noteの売上画面のスクショを使った宣伝はNG
- **「稼げる」系ワード**: 「稼げ」等のワードが入っているだけで規約違反と判断される場合がある
- **実績の過度な強調**: 過去の投資成績をことさらに強調し、将来の成果を示唆するような表現

### 4.3 noteの上場とコンテンツ規制の強化

- noteは上場を控えており（2024年に上場済み）、クリーンなプラットフォームイメージの維持に注力
- 投資系・お金儲け系・ナンパ系コンテンツが特に厳しく監視されている
- 通告なしでの記事削除、売上金の「違約金として没収」が発生する場合もある

> 出典: [note公式利用規約](https://terms.help-note.com/hc/ja/articles/44943817565465), [noteの規約違反（wonderful-wife.net）](https://wonderful-wife.net/blog/note-rule/)

## 5. コンテンツの合法的な構成方法

### 5.1 安全な構成パターン

以下のように構成すれば、金融商品取引法とnote規約の両方に抵触するリスクを最小化できる:

#### パターンA: 「AI開発の教育コンテンツ」として販売

```
タイトル例: 「Pythonで作るFX予測AIモデル - 機械学習入門から実装まで」

内容構成:
- 機械学習の基礎理論
- データ取得・前処理の方法
- モデル構築のコード解説
- バックテスト手法の解説
- Walk-Forward検証の実装

免責事項: 「本記事は技術教育を目的としたものであり、
          特定の金融商品の売買を推奨するものではありません」
```

#### パターンB: 「分析手法・フレームワーク」として販売

```
タイトル例: 「競馬AI予測モデルの構築記録 - 全開発プロセス公開」

内容構成:
- 予測モデルのアーキテクチャ解説
- 特徴量エンジニアリングの詳細
- 過去のバックテスト結果（過去データの分析として）
- モデル精度の改善プロセス

免責事項: 「本記事の内容は過去のデータ分析結果であり、
          将来の結果を保証するものではありません」
```

#### パターンC: 「実体験レポート」として販売

```
タイトル例: 「AI×ボートレース予測に1年間挑戦した全記録」

内容構成:
- 挑戦の動機と背景
- 試行錯誤の過程（失敗談含む）
- 得られた知見・学び
- データ分析の結果

免責事項: 「あくまで個人の体験記であり、
          特定の投資行動を推奨するものではありません」
```

### 5.2 必須の免責事項テンプレート

すべての有料記事に以下を含めること:

```
【重要な免責事項】
- 本記事は教育・情報共有を目的としたものです
- 特定の金融商品の売買を推奨するものではありません
- 投資判断はご自身の責任で行ってください
- 過去のデータ分析結果は将来の成果を保証しません
- 本記事の著者は投資助言・代理業の登録を行っておりません
- 投資に関する最終的な判断は、必要に応じて専門家にご相談ください
```

### 5.3 絶対にやってはいけないNG表現

| NG表現 | 安全な代替表現 |
|--------|--------------|
| 「この銘柄を買え」 | 「このような分析手法で銘柄を選定しました」 |
| 「絶対に儲かる」 | 「バックテストでは○○の結果でした」 |
| 「100%的中する」 | 「過去データでの正解率は○○%でした」 |
| 「今すぐ買うべき」 | 「このような条件のとき、過去にはこのような動きがありました」 |
| 「月収100万円達成」 | （売上公開自体がnote規約違反のため記載不可） |
| 「必ず利益が出る」 | 「期待値がプラスの手法を検証しました」 |

## 6. アカウントBAN対策

### 6.1 BANされやすいジャンルと行為

noteでBANされやすいのは以下のジャンル:
- **投資系** / **稼ぐ系** / モテ系 / アダルト系

特にリスクの高い行為:
- 投資助言に該当する内容の有料販売
- 断言系表現の使用
- 売上スクリーンショットの公開
- 銘柄推奨・買い目の直接的な提供
- 「稼げる」系のワードの多用

### 6.2 BANを防ぐための具体策

1. **売上金はこまめに振込申請する**: BANされると未振込の売上金は没収される
2. **記事のバックアップを取る**: BANされても別プラットフォームで復活可能
3. **免責事項を必ず記載する**: 全記事に投資助言ではない旨を明記
4. **断言表現を避ける**: 「〜の可能性がある」「〜と考えられる」を使用
5. **教育コンテンツとして構成する**: 「予測の結果」ではなく「予測の方法」を売る
6. **複数プラットフォームに分散する**: note一極集中のリスクを回避

### 6.3 BANされた場合の対処

- note運営への問い合わせ: 具体的な理由は教えてもらえない場合が多い
- 別プラットフォーム（Tips、Brain、WordPress+Stripe等）への移行を検討
- バックアップからコンテンツを復旧

> 出典: [noteのアカウント停止を解説（zero-pri.com）](https://www.zero-pri.com/entry/note-account-ban), [noteアカウント停止（Tips）](https://tips.jp/u/traderx/a/noteiten)

### 7.3 noteの手数料詳細

noteの手数料構成（3層構造）:

1. **事務手数料（決済手数料）**
   - クレジットカード: **5%**
   - PayPay: **7%**
   - キャリア決済: **15%**

2. **プラットフォーム利用料**（事務手数料控除後の金額に対して）
   - 有料記事/マガジン/チップ/メンバーシップ: **10%**
   - 定期購読マガジン: **20%**

3. **振込手数料**: **270円/回**

**手取り計算例:**
- 1,000円の記事（クレカ決済）: 手取り約585円（58.5%）
- 10,000円の記事（クレカ決済）: 手取り約8,280円（82.8%）

## 8. 税金・確定申告

### 8.1 確定申告が必要な基準

- **会社員の場合**: noteの年間所得（売上 - 経費）が**20万円超**で確定申告が必要
- **フリーランス/個人事業主**: 確定申告は原則必須（所得38万円超）
- **注意**: 20万円以下でも**住民税の申告は必要**

### 8.2 所得区分

- noteの収入は原則として**雑所得**
- 事業としての規模・継続性がある場合は**事業所得**も可能

### 8.3 収入の種類

以下はすべて課税対象:
- 有料記事の販売収入
- メンバーシップ（月額課金）収入
- サポート（投げ銭）収入

### 8.4 経費として計上できるもの

- パソコン・機材の購入費（一部減価償却）
- 書籍・資料の購入費
- 取材交通費
- インターネット通信費（按分）
- ソフトウェア利用料（API費用等）
- 有料データの購入費
- 勉強会・セミナー参加費

### 8.5 noteの源泉徴収

- noteは源泉徴収を行わない
- 源泉徴収票は発行されない
- 自分で売上と経費を管理し、確定申告を行う必要がある

### 8.6 売上金の確認方法

- noteのダッシュボードから「売上情報をダウンロードする」でCSVを取得
- CSVの「売上」欄を売上、「手数料」「振込手数料」欄を経費として計上

### 8.7 副業バレ対策

- 確定申告書の「住民税の徴収方法の選択」で**「自分で納付（普通徴収）」**を選択
- 副業分の住民税が自宅に届き、会社の給料から天引きされない

> 出典: [noteヘルプ - 確定申告FAQ](https://www.help-note.com/hc/ja/articles/26435214355353), [副業noteの税金（税理士解説）](https://note.com/sakata_zeirishi/n/nef1da63da2ee)

## 9. 返金トラブルと対処法

### 9.1 noteの返金制度

- 購入者は購入後**24時間以内**に返金申請が可能
- ただし、クリエイターが「**返金不可**」に設定している場合は申請不可
- 返金はnoteが審査のうえ処理する

### 9.2 返金設定の戦略

**返金可にするメリット:**
- 購入のハードルが下がる
- 信頼感が向上する
- 悪質クレームのリスク軽減

**返金不可にするメリット:**
- フリーライダー（読んで返金）を防げる
- 収益の安定性が高まる

**推奨**: 投資系コンテンツは「**返金可**」にしておく方が安全。返金不可にすると詐欺扱いされやすい。

### 9.3 トラブルを防ぐためのポイント

1. **無料部分で内容を十分に説明する**: 購入前にコンテンツの方向性がわかるようにする
2. **過度な期待を煽らない**: 「これを読めば確実に勝てる」等の表現は絶対に避ける
3. **価格に見合う量と質を提供する**: スカスカな内容は返金・クレームの原因
4. **免責事項を明記する**: 投資結果を保証しない旨を明確に
5. **更新日を記載する**: 情報の鮮度を示す

### 9.4 悪質なクレームへの対処

- noteの返金審査に委ねる（クリエイターが直接対応する必要はない）
- 記事の内容に問題がなければ、返金は認められにくい
- 消費者センターに相談されるケースもあるが、適切な免責事項があれば問題ない

> 出典: [noteヘルプ - 返金ルール](https://www.help-note.com/hc/ja/articles/360000670602)

## 10. 炎上事例と回避策

### 10.1 過去の主な炎上パターン

1. **経歴詐称**: 偽の経歴・実績でコンテンツを販売し発覚
2. **内容のパクリ**: 他者の予想やコンテンツを盗用して販売
3. **誇大広告**: 「100%的中」「必ず儲かる」等の表現
4. **的中実績の改ざん**: 過去の予想記事を削除して実績を水増し
5. **高額コンサルへの誘導**: 安価なnoteで集客し、高額サービスに誘導

### 10.2 炎上を防ぐための原則

1. **実績は正直に開示する**: 勝率・損益を含めた正確なデータ
2. **失敗談も書く**: 成功だけでなく失敗の過程も共有する
3. **謙虚な表現を使う**: 「検証中」「仮説段階」等の留保付き表現
4. **独自コンテンツであることを証明する**: 開発過程のスクリーンショット等
5. **読者との信頼関係を構築する**: 無料記事での実績 → 有料記事への自然な導線

### 10.3 noteが特に監視している表現

深津貴之氏（note CXO）が明言しているNGコンテンツ:
- 「1日30分であなたも絶対に月収100万円」系
- 優良誤認を意図した記事
- コンテンツの盗用
- 犯罪の教唆
- 「この記事を売れ」的なマルチ商品

> 出典: [深津貴之 - noteにおける情報商材について](https://note.com/fladdict/n/nf94f80fdad34)

## 11. 実行チェックリスト

### 記事公開前チェック

- [ ] 免責事項が記載されているか
- [ ] 特定の銘柄の売買を推奨する表現がないか
- [ ] 「絶対」「必ず」「100%」等の断言表現がないか
- [ ] 売上・利益のスクリーンショットを掲載していないか
- [ ] 「稼げる」「儲かる」系の煽り表現がないか
- [ ] 過去データの分析結果と将来の予測が区別されているか
- [ ] 教育・情報共有目的であることが明確か
- [ ] 無料部分で内容の方向性が十分に伝わるか
- [ ] 返金設定を確認したか
- [ ] コンテンツがオリジナルであるか（盗用・パクリなし）

### ビジネス運営チェック

- [ ] 売上金を定期的に振込申請しているか
- [ ] 記事のバックアップを取っているか
- [ ] 月間売上が100万円を超えた場合、特定商取引法の対応をしているか
- [ ] 確定申告用の売上・経費記録をつけているか
- [ ] 住民税の「普通徴収」設定を確認したか（副業の場合）
- [ ] 複数プラットフォームでの展開を検討したか

### 法的安全チェック

- [ ] コンテンツが「投資助言」に該当しない構成になっているか
- [ ] 不特定多数が随時購入可能な状態を維持しているか
- [ ] 個別の投資相談・質問対応を有償で行っていないか
- [ ] 月額課金のみ（単発購入不可）の形態になっていないか
- [ ] 必要に応じて弁護士・行政書士に相談したか
