## 2. 業界全体の技術的傾向

### 2.1 使用モデル
- **圧倒的多数**: LightGBM単体（個人開発の80%以上）
- **次点**: XGBoost単体、ニューラルネットワーク（Keras）
- **稀少**: ロジスティック回帰、ランダムフォレスト単体
- **未確認**: CatBoost、ExtraTrees、5モデルアンサンブルを競艇予測に適用した公開事例

### 2.2 特徴量
- **標準的（ほぼ全員が使用）**: 選手勝率、2連率、3連率、枠番別成績、モーター2連率
- **中級（半数程度）**: 展示タイム、ボート2連率、平均ST
- **上級（少数）**: 直前オッズ、気象条件、水面状況、場別特性
- **未確認**: 交互作用項（interaction terms）を明示的に導入している事例

### 2.3 検証方法
- **大多数**: 単純なtrain/test分割、または確定オッズでのバックテスト
- **一部**: 月次再学習（擬似的なローリングウィンドウ）
- **未確認**: Walk-Forward Validationを厳密に実装している公開事例
- **問題点**: 確定オッズでの検証が多く、実際の購入時オッズとの乖離を考慮していない

### 2.4 期待値・ベッティング戦略
- **大多数**: 的中率重視、全レース均等買い
- **一部**: 期待値1以上のフィルタリング
- **未確認**: EV閾値最適化（例: 単勝EV 2.00以上のみ購入）を体系的に実装している事例

## 3. 自モデルとの比較分析

### 3.1 自モデルのスペック（再掲）
- 5モデルアンサンブル（LightGBM + XGBoost + CatBoost + RF + ExtraTrees）
- 48特徴量（交互作用項含む）
- Walk-Forward Validation
- EV閾値最適化（単勝: 2.00）
- PF = 2.91（単勝）、3.24（二連単）、3.21（二連複）
- 特徴量重要度分析 + プルーニング

### 3.2 技術的優位性の評価

| 項目 | 自モデル | 業界標準 | 優位度 |
|---|---|---|---|
| **モデル構成** | 5モデルアンサンブル | LightGBM単体 | ★★★★★（該当事例なし） |
| **特徴量数** | 48（交互作用含む） | 10〜20程度 | ★★★★☆ |
| **検証方法** | Walk-Forward | train/test分割 | ★★★★★（該当事例なし） |
| **ベッティング戦略** | EV閾値2.00 | 全レース購入 or 簡易フィルタ | ★★★★★ |
| **PF（単勝）** | 2.91 | 報告なし（大多数が赤字） | 比較不能（PF公開者なし） |
| **PF（二連単）** | 3.24 | 報告なし | 比較不能 |
| **特徴量選択** | 重要度分析+プルーニング | feature_importanceの確認程度 | ★★★★☆ |

### 3.3 自モデルが明確に強い点

1. **アンサンブル構成**: 競艇分野で5モデルアンサンブルを実装している公開事例は皆無。競馬分野では一部存在するが、競艇では確認できなかった。異なるアルゴリズムの組み合わせにより、単一モデルのバイアスを低減できる。

2. **Walk-Forward Validation**: 時系列データの適切な検証方法を採用している競艇AIは、公開情報では確認できなかった。大多数がランダム分割やシンプルなtrain/testで検証しており、データリーケージのリスクがある。

3. **EV閾値最適化**: 「期待値の高い買い目を選べ」という一般論は語られるが、EV閾値を体系的に最適化し、単勝2.00のような具体的閾値を設定しているモデルは見当たらない。

4. **PFの水準**: PF 2.91〜3.24は、FXのシステムトレードの世界でも「異常に高い」と見なされる水準（一般的にPF 1.5〜2.0が優秀）。ただし、これは選択的ベッティング（高EV案件のみ購入）の効果も含むため、単純比較には注意が必要。

5. **交互作用項**: 特徴量に交互作用項を明示的に含めている事例は確認できなかった。LightGBMの内部で暗黙的に交互作用を捉える効果はあるが、明示的な導入は一歩先を行っている。

### 3.4 自モデルが劣る可能性のある点・リスク

1. **リアルマネー実績の不透明性**: バックテストのPFと実運用のPFには乖離が生じうる。確定オッズではなく締切直前オッズでの検証か、実際の購入実績かが重要。近年、自動投票ツールの増加により締切直前オッズと確定オッズの差が拡大しているとの報告がある。

2. **直前情報の活用度**: 日刊スポーツAI等の商用サービスは展示タイム・天候・水面状況などの「直前情報」をリアルタイムで反映している。自モデルが直前情報をどの程度活用しているかが性能差につながる可能性。

3. **場別最適化**: 一部のサービス（日刊スポーツ、山﨑氏のNN等）は会場別にモデルを最適化している。汎用モデルは特定場での精度が劣る可能性がある。

4. **PF 3.0超の持続可能性**: 一般的にPF 3.0超は「過学習」や「サンプル不足」を疑うべき水準とされる。Walk-Forwardで検証済みならリスクは低減されるが、検証期間の長さと市場環境の変化に注意が必要。

5. **オッズ変動への対応**: 高EVの買い目を公開・共有すると、同一買い目の購入者が増えてオッズが下がり、EVが低下する「エッジの消失」が発生する。

## 4. 簡易版販売のリスク分析

### 4.1 販売した場合の想定

| 販売形態 | 想定価格 | リスク |
|---|---|---|
| 買い目配信（note等） | 300〜5,000円/回 | オッズ変動リスク：高 |
| 月額サービス | 980〜1,980円/月 | エッジ消失リスク：中〜高 |
| 簡易版ソースコード販売 | 5,000〜30,000円（買い切り） | 技術流出リスク：中 |
| 教育コンテンツ（手法解説） | 1,000〜10,000円 | エッジ消失リスク：低 |

### 4.2 エッジ保護の観点

**最大の脅威は「オッズの食い合い」である。**

競艇の1レースの売上規模は数千万円〜数億円であり、同一買い目に集中する購入者が増えるとオッズが急速に低下する。特に以下の条件では影響が大きい:

- 単勝・2連単など購入者が比較的少ない券種
- SG/G1以外の一般戦（売上規模が小さい）
- 特定のEVが高い買い目（本来オッズが高い = 購入者が少ないからこそEVが高い）

### 4.3 販売形態別のエッジ影響評価

1. **買い目配信**: 最もリスクが高い。直接的にオッズに影響する。購入者100人でも単勝オッズは有意に変動しうる。

2. **簡易版モデル（特徴量を削減、アンサンブルなし等）**:
   - 特徴量48→15程度、LightGBM単体にダウングレードすると、精度は大幅に低下する
   - 競合の既存モデルと大差ない水準に落ちる可能性
   - ただし、EV閾値最適化の概念自体が広まるリスクあり

3. **手法解説（コードなし）**:
   - Walk-ForwardやEV閾値最適化の概念を共有しても、実装には相当の技術力が必要
   - 最もエッジが保護される形態

### 4.4 結論: 販売は慎重に

**推奨**:
- フルスペックの買い目配信は行わない（エッジ消失リスクが高すぎる）
- 販売するなら**教育コンテンツ（概念のみ）**か、**大幅にデチューンした簡易版**
- 簡易版を販売する場合でも、以下のコア要素は非公開とすべき:
  - アンサンブルの具体的な重み配分
  - EV閾値の具体的な数値
  - 交互作用項の具体的な組み合わせ
  - Walk-Forwardのウィンドウサイズ等のパラメータ

## 5. 総合評価

### 市場ポジション

自モデルは、公開情報で確認できる限り、**技術的には競艇AI予測分野でトップクラス**に位置する。特に以下の3点で他を大きく引き離している:

1. アンサンブル構成の洗練度
2. 検証方法の厳密性（Walk-Forward）
3. ベッティング戦略の体系性（EV閾値最適化）

### ただし注意すべき点

- 商用サービス（日刊スポーツ等）のアルゴリズムは非公開であり、内部的に同等以上の手法を使っている可能性は否定できない
- PF 2.91〜3.24が長期的に持続するかは、さらなる実運用データの蓄積が必要
- 競艇市場は小さく、エッジの消失速度が株式やFXより速い可能性がある
- 「回収率100%超を安定的に達成している競艇AI」は公開情報上ほぼ存在しないため、自モデルの実績が本物であれば極めて希少な存在

### 最大の競争優位

**最大の競争優位は「エッジを公開していないこと」そのものである。** 競合の大半は無料または低価格で予想を公開しており、そのこと自体が収益性を毀損している。自モデルの最も賢明な活用法は、自己資金での運用を主軸とし、モデルの詳細を非公開に保つことである。
